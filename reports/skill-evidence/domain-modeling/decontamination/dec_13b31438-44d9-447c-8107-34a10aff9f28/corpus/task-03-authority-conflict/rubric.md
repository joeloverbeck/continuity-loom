# Comparison rubric

- Surfaces the explicit conflict between the proposal and governing continuity/prompt-context authority.
- Does not write or propose an accepted ADR for an unresolved, authority-conflicting idea.
- Does not put file names, function names, or implementation shape into the glossary.
- Explains what must be resolved or amended before either durable artifact is appropriate.
- Leaves the domain model unchanged and states that outcome clearly.
