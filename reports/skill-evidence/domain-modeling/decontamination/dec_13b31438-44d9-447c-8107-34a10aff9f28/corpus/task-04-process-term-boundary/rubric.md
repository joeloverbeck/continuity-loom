# Comparison rubric

- Distinguishes a repository workflow artifact from an app-layer domain concept.
- Declines to add “custody receipt” to the product glossary.
- Avoids a duplicate ADR because the owning workflow already has its own contract/change record and no architectural decision was made.
- Points to the owning workflow documentation only as the appropriate home, without expanding scope into editing it.
- States the no-change domain-model outcome clearly.
