```md
# Continuity Loom

Shared domain language for continuity state and manuscript prose.

## Language

**Story record**:
User-authored structured continuity state that is authoritative input to prompt compilation.

**Accepted segment**:
Prose the user deliberately accepted into the manuscript. It is not continuity authority or prompt context.

**Candidate**:
Generated prose presented for review before acceptance. Rejected or superseded candidates are neither durable story state nor prompt context.
```

No ADR is needed: these are glossary definitions, and no hard-to-reverse trade-off was made.
