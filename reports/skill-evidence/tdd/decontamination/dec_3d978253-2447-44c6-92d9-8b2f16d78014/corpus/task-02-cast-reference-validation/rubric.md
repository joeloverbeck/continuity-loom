# Comparison rubric

- Chooses a narrow core-validation seam and begins with a failing unselected-reference test.
- Builds a matrix covering three failure modes, three cast bands, and recovery without losing existing behavior.
- Explicitly protects other reference lanes and prompt/send gating.
- Keeps docs synchronization after behavioral green rather than using docs as proof.
- Includes focused checks, regression checks, and canonical gates.
