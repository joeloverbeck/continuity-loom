- Expands through server and web only after the core behavior is green.
- Preserves draftability while proving fail-closed prompt/candidate/provider gates.
- Covers boundary transitions, explicit repair, fresh recompilation, and accepted-prose canaries.
- Defines sensible incremental commits/refactors and focused-to-canonical verification.
