# Frozen paired-trial corpus

Frozen before any semantic reading of the claimed baseline or construction of a candidate.

Each numbered directory contains an agent-facing task and an evaluator-only rubric. The tasks are synthetic reductions of real implementation runs; names and repository details are intentionally genericized, while state transitions and risk surfaces are preserved.

Agents receive only `task.md` plus one implementation-skill version. Evaluators receive both paired outputs and `rubric.md`, with version labels randomized.
